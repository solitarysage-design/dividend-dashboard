📖 この画面の読み方
「今の株価で買っていいか」 を判定しています。適正価格（過去の利益・配当実績から算出）と現在株価を比較し、割安度を判断します。
● BUY_NOW = 割安。買い時
● BUY_SMALL = やや割安。少額で打診買い
● WAIT = 適正〜割高。もう少し待ち
● AVOID = 監査NGなど。買い不適格
🔄 判定フロー (判定はこの順番で行われます)
① 監査ゲート → FAIL/ALERT → 即AVOID
② 利回り帯 → 過去5年平均との比較でスコア加減点
③ PER帯 → 過去5年平均との比較でスコア加減点
④ 安全域(MOS) → 適正価格からの乖離率でスコア加減点
⑤ カタリスト → 割安解消の見込みがあるか
⑥ 市場レジーム → 日経全体の過熱/売られすぎ
⑦ テクニカル → 週足RSI・52週高値・200日MA
→ 合計スコア: ≥35=BUY_NOW / ≥15=BUY_SMALL / ≥-5=WAIT / <-5=AVOID
適正価格 = 利益ベース(EPS×平均PER)と配当ベース(DPS÷平均利回り)の業種別加重平均
MOS(安全余裕率) = (適正価格−現在株価)÷適正価格×100。プラス=適正価格より安い（お買い得）、マイナス=割高。+20%以上=割安 / 0〜+19%=やや割安 / マイナス=割高
買いゾーン = ① 打診買い=適正価格(FV) / ② 標準買い=FV×0.9(10%引) / ③ 強気買い=FV×0.75(25%引) ※ボラティリティ係数で調整
増配プレミアム = 連続増配5年以上+監査PASS → 適正価格を+3〜5%引き上げ (増配10年で+5%, 加速中なら追加+3%)
テクニカル = 週足RSI・52週高値乖離・200日MA乖離でファンダ判定を補強。テクニカルのみで判定は覆りません
